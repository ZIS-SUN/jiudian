# 酒店管理系统

## 项目结构
```
hotel-management/
├── backend/          # Spring Boot 后端
├── frontend/         # Vue3 前端  
├── docs/            # 项目文档
└── README.md        # 项目说明
```

## 技术栈
- 前端：Vue3 + Element-Plus + Axios
- 后端：Spring Boot 3 + MyBatis-Plus + MySQL
- 数据库：MySQL 8.0

## 开发环境
- Java 17+
- Node.js 16+
- MySQL 8.0
